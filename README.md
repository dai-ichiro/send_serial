# WinForm シリアル通信ツール

C# Windows Forms (.NET Framework 4.8) で作成された、シンプルなシリアルポート通信アプリケーションです。
指定したシリアルポートに接続し、ボタン操作で1バイトのデータを送信することができます。

## 機能

* **ポート検出**: システム上で利用可能なシリアルポートを自動的に検出し、リストに表示します。
* **接続**: 選択したシリアルポートに接続します。
  * 固定設定: BaudRate 9600, DataBits 8, Parity None, StopBits 1
* **データ送信**: UI上のボタンをクリックすることで、そのボタンに割り当てられた数値（0-255の範囲）を1バイトのバイナリデータとして送信します。
* **エラー処理**: ポートが既に使用中である場合や、存在しない場合の適切なエラーハンドリングを行います。

## 必要要件

* Windows OS
* .NET Framework 4.8
* Visual Studio 2019 またはそれ以降（開発・ビルド時）

## 使用方法

1. アプリケーションを起動します。
2. 画面上のドロップダウンリストから、接続したいCOMポートを選択します。
3. 「接続」ボタン（または類するボタン）をクリックしてポートを開きます。
   * 接続に成功すると、接続ボタンは無効化されます。
4. 数値が記載されたボタンをクリックすると、その数値に対応する1バイトのデータがシリアルポート経由で送信されます。
5. アプリケーションを終了すると、自動的にポートが閉じられます。
